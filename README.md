# Arise

**Arise** is a minimalist discipline companion built for people who want to follow a routine without unnecessary distractions.

Unlike productivity apps packed with features, Arise focuses on one goal: **show up every day.**

---

## Philosophy

- Minimal UI
- Zero distractions
- Fast
- Offline-first
- No ads
- No subscriptions
- No unnecessary customization

Discipline should not depend on motivation.
